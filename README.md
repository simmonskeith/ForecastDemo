# Forecast Demo

A simple React app that fetches the weather forecast from weather.gov's api given a US zip code.

Once running, access it via http://localhost:3000

Includes unit tests and Playwright end-to-end tests as well as Docker containerization.

### Useful Commands

`npm run test` - runs the unit tests
`npm run test:e2e` - runs the Playwright tests
`npm run dev` - runs the app

`docker compose up -d` - run the app in a Docker container
`docker compose down` - stop the docker container
